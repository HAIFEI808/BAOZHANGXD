## 五、关键数据结构与接口描述

### 5.1 RTC_DateTime_t (`rtc_time.h`)
```c
typedef struct {
    uint8_t year;   /* 0-99, 对应 2000-2099 年 */
    uint8_t month;  /* 1-12 */
    uint8_t date;   /* 1-31 */
    uint8_t hour;   /* 0-23, 24小时制 */
    uint8_t min;    /* 0-59 */
    uint8_t sec;    /* 0-59 */
} RTC_DateTime_t;
```
| 接口 | 说明 |
|---|---|
| `void RTC_Get_DateTime(RTC_DateTime_t *dt)` | 从 RTC 硬件读取完整日期时间 |
| `void RTC_Set_DateTime(RTC_DateTime_t *dt)` | 写入日期时间到 RTC 硬件（用于校准） |
| `void RTC_Get_CurrentTime(uint8_t *h, uint8_t *m, uint8_t *s)` | 仅读取时分秒（兼容旧接口） |

### 5.2 CST816D 触控接口 (`cst816d.h`)
| 接口 | 说明 |
|---|---|
| `uint8_t CST816D_Init(void)` | 初始化触控芯片，I2C 验证 CHIP ID，返回 0 成功 |
| `uint8_t CST816D_GetGesture_Debounced(uint8_t sample_ms)` | 消抖手势读取：连续 3 次采样一致才返回手势值，否则返回 NONE |
| `uint8_t CST816D_GetGesture(void)` | 直接读取手势寄存器（无消抖） |
| `void CST816D_GetXY(uint16_t *x, uint16_t *y)` | 读取当前触摸坐标 |
| `uint8_t CST816D_IsPressed(void)` | 读取触摸点数量，返回是否按下 |

触控检测流程: CST816D INT(PB10) 下降沿 → EXTI15_10_IRQHandler → vTaskNotifyGiveFromISR(touchTask) → touchTask 唤醒 → 3 次消抖采样 → xTaskNotify + osEventFlagsSet → sysTask 处理

### 5.3 I2S 闹钟接口 (`i2s_alarm.h`)
```c
typedef struct {
    uint8_t hour;    /* 0-23 */
    uint8_t min;     /* 0-59 */
    uint8_t enabled; /* 0=关, 1=开 */
} AlarmTime_t;
```
| 接口 | 说明 |
|---|---|
| `void I2S_Alarm_Init(void)` | 初始化 I2S2+DMA1_Stream4 外设，预计算 1kHz 正弦波 PCM |
| `void I2S_Alarm_Start(void)` | 启动闹钟提示音 (DMA 循环播放) |
| `void I2S_Alarm_Stop(void)` | 停止闹钟提示音 |
| `uint8_t I2S_Alarm_IsPlaying(void)` | 查询是否正在播放 |

**闹钟逻辑** (sysTask 中实现):
- 触发: `alarm_time.enabled && 时间匹配 && sec<5 && 未播放` → `I2S_Alarm_Start()`
- 自动停止: `sec>=5` → `I2S_Alarm_Stop()`
- 触控停止: 任意触控手势 → `I2S_Alarm_Stop()`

### 5.4 FreeRTOS 通信对象 (`main.c`)
| 对象 | 类型 | 用途 |
|---|---|---|
| `lvgl_mutexHandle` | `osMutexId_t` | 保护所有 LVGL API 调用，guiTask/sysTask 双向持锁 |
| `rtcQueueHandle` | `osMessageQueueId_t` | rtcTask → sysTask，传输 `RTC_DateTime_t`，容量 4 |
| `sysEventHandle` | `osEventFlagsId_t` | `SYS_EVENT_TOUCH`(0x01) 触控唤醒 / `SYS_EVENT_SLEEP`(0x02) 休眠协同 |
| `sleepTimerHandle` | `osTimerId_t` | 10s 单次 osTimer，超时回调置位 SYS_EVENT_SLEEP；触控时 restart |
| `homeTaskHandle` | `osThreadId_t` | sysTask 句柄，供 touchTask 发送任务通知 |

### 5.5 LVGL 默认字体依赖
| 字体 | 使用位置 |
|---|---|
| `lv_font_montserrat_24` | 主表盘时钟标签、设置页冒号分隔符 |
| `lv_font_montserrat_18` | 设置页标题、时/分 roller |
| `lv_font_montserrat_14` | 主表盘日期标签 |
